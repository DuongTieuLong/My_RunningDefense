# Running Defense
Overview
Running Defense is a roguelike survival tower defense game where the player must protect a central tower against waves of enemies.
The objective is to survive as long as possible — if the tower is destroyed, the game ends.
Players defeat enemies to gain experience, unlock new weapons, and upgrade both their character and the tower. Between runs, players can purchase equipment to improve their base stats and progression.

# Core Features
# Gameplay Loop
  Choose a map and survive enemy waves
  Defend the tower while fighting enemies
  Gain experience → level up → choose upgrades
  Earn currency → buy equipment → improve future runs
  
# Combat & Progression
  Player gains experience from killing enemies
  Level-up system allows:
    Unlocking new weapons
    Upgrading existing weapons
  Multiple weapon types:
    Rotating weapons (area damage on contact)
    Auto-shoot weapons
    Auto-stab/melee weapons
  Each weapon has multiple upgrade paths
  
# Tower System
  Tower has:
    Health
    Level
    Upgradable stats
  Tower can attack enemies automatically
  Tower gains experience and can be upgraded during gameplay
  
# Enemy System
  Multiple enemy types:
    Melee enemies
    Ranged enemies
    Elite enemies (enhanced stats)
    Boss enemies (unique per map)
  3 different maps with distinct environments and enemy sets
  
# Economy & Equipment
  Currency earned from:
    Killing enemies
    Completing achievements
  Equipment system:
    Helmet, Armor, Boots, Ring
    Provides stat bonuses (HP, damage, movement speed)
  Equipment persists across runs
  
# Supply Drop System
  Random drops appear on the map
  Effects include:
    Heal player
    Heal tower
    Area explosion damage around tower
    
# Achievement System
  Tracks player progression:
    Longest survival time
    Total currency earned
    Purchased equipment
  Persistent progression system
